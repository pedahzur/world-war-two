## Manuscript structure and writing order

### Recommended writing sequence

1. **Theoretical Framework (Section II)** — write first to lock the analytical lens
2. **British Cases (Section III)** — strongest evidence, most cases
3. **American Cases (Section IV)** — follows naturally from British via diffusion
4. **Cross-National Comparison (Section V)** — synthesizes III and IV
5. **Alternative Explanations (Section VI)** — stress-tests the framework
6. **Introduction (Section I)** — write last (or second-to-last) when the argument is fully crystallized
7. **Conclusion (Section VII)** — write last
